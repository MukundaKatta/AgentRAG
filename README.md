Modular RAG pipeline for AI agents. Chunking, embedding, retrieval, and reranking with pluggable vector stores. Built for agenic workflows. AgentRAG gives you a focused, inspectable implementation of that idea.

AgentRAG exists to make this workflow practical. Modular rag pipeline for ai agents. chunking, embedding, retrieval, and reranking with pluggable vector stores. built for agenic workflows. It favours a small, inspectable surface over sprawling configuration.
